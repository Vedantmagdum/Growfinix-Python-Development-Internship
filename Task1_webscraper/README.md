# 🏠 Property Web Scraper 

A beginner-friendly Python web scraping project that extracts property listings from a demo HTML webpage using Requests and BeautifulSoup and saves the data into a CSV file.

## Features

- Scrape property titles
- Extract prices
- Extract locations
- Save data to CSV
- Simple and beginner-friendly

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- HTML
- CSV

## Project Structure

```text
Property-Web-Scraper/
├── property_listings.html
├── property_listings.csv
├── scraper.py
├── README.md
├─  style.css
```

## Installation

```bash
pip install request
pip install beautifulsoap
```

## Run the Project

1. Open `property_listings.html` with Live Server.
2. Copy the local URL (for example `http://127.0.0.1:5500/property_listings.html`).
3. Update the URL in `scraper.py` if needed.
4. Run:

```bash
python scraper.py
```

## Output

A file named `property_listings.csv` will be generated.
